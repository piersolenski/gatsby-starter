# Gatsby Starter

A simple boilerplate for kickstarting [Gatsby](https://www.gatsbyjs.org/) projects.

Check out the [demo](https://gatsby-starter-template.netlify.com)!

## Features

- Gatsby
- Styled Components + Polished
- Prettier + ESLint integration
- SVGR

### Installation
